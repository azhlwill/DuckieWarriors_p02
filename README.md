## Description
  - We will create a recipe sharing and viewing platform that allows users to share and explore recipes from a range of cuisines. Users will be able to browse information about recipes on sample pages as well as pages put up by other users. Users can contribute their own recipes to the site. We will be using external APIs to load the images and information for the premade recipes.
